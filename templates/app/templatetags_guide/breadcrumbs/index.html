Hiển thị luồng đi của page
{% breadcrumb breadcrumb_list=breadcrumb_data %}

Mẫu data truyền vào option breadcrumb_list:

              
breadcrumb_data = [
    {
      'label': 'Home',
      'url': reverse("Screen:home")
    },
    {
      'label': 'Template Tags Guide',
      'url': reverse("_Extract_Tag:list_guide")
    },
    {
      'label': 'Breadcrumbs'
    }
]

Mô tả cho data

Name data Description Type Data Example Lưu ý
label Nội dung text hiển thị của mỗi item Breadcrumbs text 'label': 'Home'
url Truyền vào URL muốn điều hướng URL text 'url': reverse("Screen:home")
  • Cần import plugin reverse để render URL sang HTML đúng chuẩn
    from django.urls import reverse
  • Khi không truyền url default item đó được active. Khi active thì không click và điều hướng sang page khác được.
