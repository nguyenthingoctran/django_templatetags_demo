Autoescape
Dùng để show đoạn text chứa HTML
Demo 1
{% autoescape on %}

{{ autoescape }}

{% endautoescape %}
BE

Truyền data vào HTML như này

            
data = '{{ autoescape }}'
FE

Trường hợp khi bật ở chế độ "on"

              
{% autoescape on %}
  <h3>{{ autoescape }}</h3>
{% endautoescape %}

Note: Trường hợp này có thể dùng cú pháp này để thay thế. Kết quả hiển thị tương tự 

              
<h1>{{ autoescape|escape }}h1>
Demo 2
{% autoescape off %}

{{ autoescape }}

{% endautoescape %}
BE

Truyền data vào HTML như này

            
data = '{{ autoescape }}'
FE

Trường hợp khi bật ở chế độ "off"

              
{% autoescape off %}
  <h3>{{ autoescape }}</h3>
{% endautoescape %}

Note: Trường hợp này có thể dùng cú pháp này để thay thế. Kết quả hiển thị tương tự 

              
<h1>{{ autoescape|safe }}h1>
