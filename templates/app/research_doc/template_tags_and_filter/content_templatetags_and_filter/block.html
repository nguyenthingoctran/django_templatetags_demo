Block
Dùng để ghi đè mã code. Tags này chỉ sử dụng ở HTML code.
FE

Tại trang main_layout.html đặt 1 block giữ chỗ để tuỳ thay đổi content. Với blockname đặt tuỳ chỉnh

              
{% block blockname %}{% endblock %}

Tại trang extend main_layout ghi đè bằng content tuỳ biến

                
{% extends 'main_layout.html' %}
{% block userinfo %}
  <h2>John Doeh2>
  <p>Explorer of life.</p>
{% endblock %}
