Cú pháp

                
{{ value|add:valuetoadd }}
1. Đối với number thì + thêm vào như 1 phép toán

Render HTML

                    
{% with x=10 %}
{{ x|add:"10" }}
{% endwith %}

Kết quả

{% with x=10 %} {{ x|add:"10" }} {% endwith %}
2. Đối với string/character thì nối vào

Render HTML

                    
{% with x='Hello ' %}
{{ x|add:"Kitty" }}
{% endwith %}

Kết quả

{% with x='Hello ' %} {{ x|add:"Kitty" }} {% endwith %}
3. Đối với list thì append vào

Mẫu data

                    
context = {
  'fruits': ['Apple', 'Banana', 'Cherry'],   
  'vegetables': ['Asparagus', 'Broccoli', 'Carrot'],
}

Render HTML

                    
{{ fruits|add:vegetables }}

Kết quả

{{ data.fruits|add:data.vegetables }}
