{{ value|add:valuetoadd }}

Đối với number thì + thêm vào như 1 phép toán

Demo: {% with x=10 %} {{ x|add:"10" }} {% endwith %} 
                    
{% with x=10 %}

Đối với string/character thì nối vào

Demo: {% with x='Hello ' %} {{ x|add:"Kitty" }} {% endwith %} 
                    
{% with x='Hello ' %}
{{ x|add:"Kitty" }}
{% endwith %}

Đối với list thì append vào

Demo: {{ filter_add.fruits|add:filter_add.vegetables }} 
                    
class Research_Doc(TemplateView):
    ...
    
    def research_doc_filter_add(self):
      context = {
        'fruits': ['Apple', 'Banana', 'Cherry'],   
        'vegetables': ['Asparagus', 'Broccoli', 'Carrot'],
      }
      return context
      
    def get_context_data(self, **kwargs):
      ...
      data['filter_add'] = self.research_doc_filter_add()
      return data
                    
                  
                    
{{ filter_add.fruits|add:filter_add.vegetables }}
