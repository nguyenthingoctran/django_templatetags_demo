Đối với number thì + thêm vào như 1 phép toán

                    
{% with x=10 %}
{{ x|add:"10" }}
{% endwith %}
Kết quả: {% with x=10 %} {{ x|add:"10" }} {% endwith %}

Đối với string/character thì nối vào

                    
{% with x='Hello ' %}
{{ x|add:"Kitty" }}
{% endwith %}
Kết quả: {% with x='Hello ' %} {{ x|add:"Kitty" }} {% endwith %}

Đối với list thì append vào

                    
context = {
  'fruits': ['Apple', 'Banana', 'Cherry'],   
  'vegetables': ['Asparagus', 'Broccoli', 'Carrot'],
}
                    
                  
                    
{{ fruits|add:vegetables }}
Kết quả: {{ data.fruits|add:data.vegetables }}
