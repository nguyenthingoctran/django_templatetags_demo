1. Thêm 's' vào nếu là số nhiều. Đây là trường hợp default khi không có có yêu cầu cụ thể text.

Data mẫu

        
data = [0, 1, 2]

Render HTML

        
{% for x in data %}
  I have {{ x }} dog{{ x|pluralize }}. 

{% endfor %}

Kết quả

{% for x in data %} I have {{ x }} dog{{ x|pluralize }}.
{% endfor %}
2. Thêm text "es" hoặc bất kỳ text nào vào cuối từ cần thêm.

Data mẫu

        
data = [0, 1, 2]

Render HTML

        
{% for x in data %}
  You have {{ x }} kiss{{ x|pluralize:'es' }} left<br/>
{% endfor %}

Kết quả

{% for x in data %} You have {{ x }} kiss{{ x|pluralize:'es' }} left
{% endfor %}
3. Thêm text chỉ định vào 2 trường hợp bằng 1 và lớn hơn 1.
