2. Đối số âm

Render HTML với float format là số âm

        
{{ 7.122489|floatformat:-2 }} <br/>
{{ 7.1|floatformat:-2 }} <br/>
{{ 7|floatformat:-2 }}

Kết quả

{{ 7.122489|floatformat:-2 }}
{{ 7.1|floatformat:-2 }}
{{ 7|floatformat:-2 }}
