cycle
Tạo list tương ứng mapping với list data trả về
Demo 1
    {% for x in cycle %}
  • {{ x }}
    • {% endfor %}
BE

Truyền data vào HTML như này

              
class Research_Doc(TemplateView):
    ...
    def research_doc_cycle(seft):
      data = ['Apple', 'Banana', 'Cherry', 'Orange', 'Durian']
      return data
      
    def get_context_data(self, **kwargs):
      ...
      data['cycle'] = self.research_doc_cycle()
      return data
FE
                
<ul>
{% for x in cycle %}
  <li style="color:{% cycle 'red' 'green' 'blue' 'pink' %}">
    {{ x }}
  </li>
{% endfor %}
</ul>

Note:

  • List cycle có thể sẽ quay lặp lại từ đầu khi đến cuối chuỗi

Demo 3 - Reset cycle
    {% for x in cycle %} {% cycle 'lightblue' 'pink' 'yellow' 'coral' 'grey' as bgcolor silent %} {% if forloop.counter == 3 %} {% resetcycle %} {% endif %}
  • {{ x }}
    • {% endfor %}
BE

Truyền data vào HTML như này

              
class Research_Doc(TemplateView):
    ...
    def research_doc_cycle(seft):
      data = ['Apple', 'Banana', 'Cherry', 'Orange', 'Durian']
      return data
      
    def get_context_data(self, **kwargs):
      ...
      data['cycle'] = self.research_doc_cycle()
      return data
FE
                
<ul>
  {% for x in cycle %}
    {% cycle 'lightblue' 'pink' 'yellow' 'coral' 'grey' as bgcolor silent %}
    {% if forloop.counter == 3 %}
      {% resetcycle %}
    {% endif %}
    <li style='background-color:{{ bgcolor }}'>
      {{ x }}
    </li>
  {% endfor %}
</ul>

Note:

  • Sử dụng resetcycle nếu muốn reset lại list cycle
