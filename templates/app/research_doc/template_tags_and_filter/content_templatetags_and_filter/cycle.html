cycle
Tạo list tương ứng mapping với list data trả về
Demo 1
    {% for x in cycle %}
  • {{ x }}
    • {% endfor %}
BE

Truyền data vào HTML như này

              
class Research_Doc(TemplateView):
    ...
    def research_doc_cycle(seft):
      data = ['Apple', 'Banana', 'Cherry', 'Orange', 'Durian']
      return data
      
    def get_context_data(self, **kwargs):
      ...
      data['cycle'] = self.research_doc_cycle()
      return data
FE
                  
<ul>
{% for x in cycle %}
  <li style="color:{% cycle 'red' 'green' 'blue' 'pink' %}">
    {{ x }}
  </li>
{% endfor %}
</ul>

Note:

  • List cycle có thể sẽ quay lặp lại từ đầu khi đến cuối chuỗi

Demo 2
    {% for x in cycle %}
  • {{ x }}
    • {% endfor %}
BE

Truyền data vào HTML như này

              
class Research_Doc(TemplateView):
    ...
    def research_doc_cycle_color(seft):
      data = 'springgreen'
      return data
      
    def get_context_data(self, **kwargs):
      ...
      data['cycle_color'] = self.research_doc_cycle_color()
      return data
FE
                  
<ul>
{% for x in cycle %}
  <li style="color: {% cycle 'red' cycle_color 'blue' 'pink' %}">
    {{ x }}
  li>
{% endfor %}
</ul>

Note:

  • Có thể truyền biến vào cycle
