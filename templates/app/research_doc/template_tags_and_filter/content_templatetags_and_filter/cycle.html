cycle
Tạo list tương ứng mapping với list data trả về
Demo 1
    {% for x in cycle %}
  • {{ x }}
    • {% endfor %}
Demo 2
    {% for x in cycle %}
  • {{ x }}
    • {% endfor %}
BE

Truyền data vào HTML như này

          
data = ['Apple', 'Banana', 'Cherry', 'Orange']
FE

Trường hợp khi bật ở chế độ "on"

              
<ul>
{% for x in cycle %}
  <li style="color:{% cycle 'red' 'green' 'blue' 'pink' %}">
    {{ x }}
  </li>
{% endfor %}
</ul>

Note:

  • List cycle có thể sẽ quay lặp lại từ đầu khi đến cuối chuỗi
