extends
Sử dụng extends để kế thừa và mở rộng từ parent template. Và dùng block tags để giữ chỗ cho các template con hiển thị
FE
Đối với file main.html 
          
<html>
  <body>

  <h1>Welcome</h1>

  {% block mymessage %}
  {% endblock %}

  </body>
</html>
Đối với file template.html 
          
{% extends 'mymaster.html' %}

{% block mymessage %}
  <p>This page has a master page</p>
{% endblock %}
